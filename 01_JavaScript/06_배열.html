배열

자바스크립트에서는 변수선언시 별도의 자료형 지정이 없기 때문에
모든 자료형의 값들을 하나의 배열공간에 담을 수 있다.(자바의 컬렉션과 유사)

* 배열의 선언

배열 선언시 배열의 크기를 지정한체로 선언하거나, 크기 지정안해도 됨(크기에 제약이 없음)

* Array객체의 메소드

1) 배열.indexOf(찾을요소) : 요소가 위치해있는 인덱스를 리턴


2) 배열.consat(배열, ...) : 여러개의 배열을 결합할 때 사용

3) 배열.reverse() : 배열에 담긴 값들을 역순으로 바꿔주는 메소드

4) 배열.sort() : 배열의 오름차순 정렬시켜주는 메소드

5_1) 배열.push(추가할요소) : 배열의 맨 뒤 요소를 추가하고 배열의 크기 반환
5_2) 배열.pop() : 배열의 맨 뒤 요소 제거하고 제거된 요소 반환

6_1) 배열.unshift(추가할 요소) : 배열의 맨 앞에 요소 추가하고 배열의 크기 반환
6_2) 배열.shift() : 배열의 맨 앞 요소 제거하고 제거된 요소 반환

7_1) 배열.slice(시작, 끝) : 배열안의 요소들을단지 추출해보는 메소드
7_2) 배열.splice(시작, 제거수[, 추가할 요소]) : 배열의 요소 추출해서 제거 및 추가

8) 배열에 담긴 값들을 하나의 "문자열"로 합쳐서 반환시켜주는 메소드
배열.toString() / 배열.join([구분자])
